# ShareFile
file sharing to the world !

- no occupy cloud storage.
- no file size limit.
- use your local tunnel to share files

<img src="screenshot/demo.gif" width="600">

## Install
```bash
npm install -g sharefile
```

## Usage
Generate share URL link.

```bash
# sharefile [your file]
# ex:

sharefile demo.png
```

> tip: `ctrl+c` to break command line
