# ShareFile
file sharing to the world !

<img src="screenshot/demo.gif" width="600">

## Install
```bash
npm install -g sharefile
```

## Usage
Generate share URL link.

```bash
# sharefile [your file]
# ex:

sharefile demo.png
```
